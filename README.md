1. Project Title / Headline

📊 Meta Ad Performance Dashboard: Facebook & Instagram Campaign Insights

An interactive Power BI dashboard built to track, analyze, and optimize Meta (Facebook & Instagram) ad campaign performance — covering reach, engagement, conversions, and audience demographics.

2. Short Description / Purpose

The Meta Ad Performance Dashboard is a visually engaging and analytical Power BI report designed to help marketing teams monitor campaign performance across impressions, clicks, engagement, and purchases. The dashboard focuses on highlighting key funnel metrics, audience demographics (age, gender, geography), and ad-type performance comparisons. This tool is intended for use by marketing managers, social media analysts, and growth teams who need to make fast, data-driven decisions on budget allocation and targeting.

3. Tech Stack

The dashboard was built using the following tools and technologies:

Power BI Desktop

Main data visualization platform used for report creation.

Power Query

Data transformation and cleaning layer for reshaping and preparing the data.

DAX (Data Analysis Expressions)

Used for calculated measures like CTR, Engagement Rate, Conversion Rate, and Purchase Rate.

Data Modeling

Relationships established among tables (campaigns, comments, demographics, and ad performance) to enable cross-filtering and aggregation.

File Format

.pbix for development, .png for dashboard previews.

4. Data Source

The dataset used for this dashboard contains simulated/sample Meta Ads performance data, including:

Campaign-level metrics

Impressions, Clicks, Shares, Comments, Purchases, Engagements.

Audience demographic data

Age, gender, country.

Time-based engagement data

Weekly trend, hourly comment activity, monthly breakdown.

Ad-type performance data

Carousel vs. Image — Impressions, Clicks, CTR, Purchase Rate, Engagement Rate, Conversion Rate.

5. Features & Highlights

Real-time KPI Tracking

Impressions, Clicks, CTR, Engagement Rate, Conversion Rate, and Purchase Rate calculated dynamically using DAX.

Dynamic Measure Selector

Lets users switch the metric being visualized (e.g., Comments) without duplicating visuals.

Audience Demographic Breakdown

Donut chart for gender split and histogram for age distribution to evaluate targeting accuracy.

Geographic Engagement Mapping

World map visual showing comment volume by country for global reach analysis.

Trend Analysis

Weekly stacked bar chart and hourly line chart to identify peak engagement periods for better ad scheduling.

Ad-Type Performance Comparison

Matrix table comparing Carousel vs. Image ads across CTR, Purchase Rate, Engagement Rate, and Conversion Rate.

Interactive Slicers

Filter by Campaign Name, Target Interest, and Month for granular, self-service analysis.

Budget Visibility

Total Budget and Average Budget per Campaign cards to tie performance directly to spend.
