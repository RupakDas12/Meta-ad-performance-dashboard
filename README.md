# 📊 Meta Ad Performance Dashboard

### Facebook & Instagram Campaign Performance Analysis using Power BI
---

## 📌 Project Overview

The **Meta Ad Performance Dashboard** is an interactive Power BI report built to analyze and optimize **Facebook & Instagram advertising campaigns**. It provides marketing teams with actionable insights into campaign performance by tracking impressions, clicks, engagement, conversions, audience demographics, and advertising efficiency.

Designed for **marketing managers, digital analysts, and growth teams**, the dashboard enables data-driven decisions regarding campaign optimization, audience targeting, and budget allocation.

---

## 🎯 Objectives

* Monitor overall advertising performance
* Measure campaign effectiveness using KPIs
* Analyze audience demographics
* Compare different ad formats
* Identify the best-performing campaigns
* Optimize advertising spend using data insights

---

| Technology           | Purpose                               |

---

## 📂 Data Source

This project uses a **simulated Meta Ads dataset** containing campaign-level performance metrics.

### Dataset includes:

* Campaign Name
* Ad Type
* Impressions
* Clicks
* Engagements
* Comments
* Shares
* Purchases
* Campaign Budget
* Audience Age
* Audience Gender
* Country
* Date & Time
* Target Interest

---

## 📈 Key Performance Indicators (KPIs)

The dashboard dynamically calculates:

* 📌 Total Impressions
* 👆 Total Clicks
* 💬 Total Comments
* ❤️ Total Engagements
* 🛒 Total Purchases
* 📊 Click Through Rate (CTR)
* 📈 Engagement Rate
* 🎯 Conversion Rate
* 💰 Purchase Rate
* 💵 Total Campaign Budget
* 📉 Average Budget per Campaign
